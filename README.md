# Product API Project
 Жоба атауы:
Product API
 Мақсаты:
Бұл жобаның мақсаты – өнімдер туралы ақпаратты басқаруға арналған REST API құру. API арқылы өнімдерді қосу, қарау, жаңарту және өшіру мүмкіндігі бар.
 Қолданылған технологиялар:
- Python
- Flask
- JSON
- REST API
 Функционал:
- GET /products → барлық өнімдерді көру
- GET /products/<id> → бір өнімді көру
- POST /products → жаңа өнім қосу
- PUT /products/<id> → өнімді жаңарту
- DELETE /products/<id> → өнімді өшіру
 Жұмыс принципі:
API клиенттен сұраныс қабылдайды және JSON форматында жауап қайтарады. Барлық деректер уақытша жадта сақталады.
Тестілеу:
Postman немесе браузер арқылы тексерілді.
 Қорытынды:
Бұл жоба REST API қалай жұмыс істейтінін түсінуге көмектесті және backend дағдыларын дамытуға мүмкіндік берді.
